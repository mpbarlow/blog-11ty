About me

Hello, my name is Matt Barlow. I’m a software developer based in the UK. I’m onstensibly a web developer, but I work on all sorts of stuff.

My bread and butter has typically been PHP (particularly Laravel projects—to which I once contributed a small but useful feature), but these days I do just as much Go, TypeScript, and Python at work as I do PHP.

I’ve worked on Objective-C and Swift iOS apps both professionally and for fun. I’ve led the migration of a large system from a ragtag collection of VPSs powered by Bash scripts and hope, to a Kubernetes cluster with CI/CD. I spent a couple of months helping optimise a data ingestion pipeline written entirely in Snowflake stored procedures, and I once made a clone of Wordle in Lua for the Playdate handheld console. I’d love to get deep into a Lisp, but I can never think of a project to build with it.

More than all that though, I enjoy long walks with my wife and dog, old video games, cars, and movies with the lights off and the sound up.

I’m not actively looking for work, but if you’re building something cool please feel free to reach out on Mastodon.

About the blog

This blog is built with the excellent Eleventy, and hosted on GitHub Pages.

Everything is proudly constructed with plain old web technology: hand-written HTML and CSS, with no frameworks and no client-side JavaScript. My goal is for the site to be fast, accessible, and privacy-preserving for everyone. It should look great in your browser’s reading mode, or your RSS reader. This website collects no analytics and stores no cookies.

Most importantly: every word I write is my own thoughts, and every line of code committed is my own work. No LLMs, no slop, no engagement bait.

Thanks for reading!
